# Third-Party Notices

The current Visualize Mode ships with procedural in-repo geometry only.

- No third-party GLTF models are bundled at this time.
- No third-party texture packs are bundled at this time.

If local assets are added later under `web/public/visualize/models` or `web/public/visualize/textures`, add attribution entries here with:

- Asset name
- Author
- License
- Source URL
